


## init set up
```
$ heroku plugins:install heroku-docker
$ heroku docker:init
```

## Local development
```
$ docker-compose build
$ docker-compose up
$ docker-compose down
```

## deployment
```
$ heroku create
$ heroku docker:release
$ heroku open

```
